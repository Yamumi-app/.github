# Yamumi

**Компактный плеер Яндекс Музыки для тех, кто занят делом.**

Yamumi позволяет слушать музыку в фоне и не превращать её в отдельную задачу.  
Минималистичный интерфейс, отображение поверх окон — всё сделано так, чтобы плеер оставался рядом, но не отвлекал.

[Попробовать Yamumi](https://yamumi.myrt.co)
